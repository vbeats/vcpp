## vcpp

充电桩 边缘设备服务

- [ ] 云快充 1.5 ~ 2.x
- [ ] OCPP 1.6 ~ 2.x

## 端口

```
云快充: 1.4 - 淘汰
云快充: 1.5 - 9847
云快充: 1.6 - 9848
云快充: 1.7 - 9849
云快充: 1.8 - 9850
云快充: 2.0 - 9851
```

## 桩号

云快充: 14位长整数

## 平台端消息通信 MQTT

> 桩 == 平台 MQTT 消息通信

- 平台指令 topic: vcpp/cmd/{充电桩号}
- 设备状态 / 数据 topic: vcpp/data/{充电桩号}

多实例部署时, 每个实例指定不同的`vcpp.mqtt.client-id`
